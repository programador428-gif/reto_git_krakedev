# Reto Git - KakreDev

Este repositorio contiene el ejercicio práctico para el dominio de comandos básicos de Git.

---
## Comandos Practicados
1. `git clone` - Clonación del proyecto.
2. `git add` - Preparación de archivos.
3. `git commit` - Confirmación de cambios.
4. `git push` - Sincronización con el remoto.
